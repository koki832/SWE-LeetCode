# SWE-LeetCode


## 学習方法
標準的な学習サイクルに基づき、以下の3ステップで進めています。

---

### **Step 1：基礎の構築**
* **思考（5分）**: まずは自力で解法を考えます。
* **解答の確認**: 分からなかった場合は一度答えを確認し、ロジックを理解します。
* **白紙からの実装**: 答えを隠した状態でコードを最初から書き、正解（Accepted）になるまで進めます。

### **Step 2：コードの洗練**
* **自己整形**: 自分のコードをより読みやすく整形します。
* **知見の導入**: 他のユーザーのコードやコメントを確認し、効率的な解法やベストプラクティスを反映させて再整形します。
* **動作確認**: 再整形後、改めて正解することを確認します。

### **Step 3：定着の確認**
* **タイムトライアル**: その問題を**「10分以内に迷わず正解のコードが書ける」**ことを最終確認します。
